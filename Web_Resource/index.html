Об операционной системе
elementary OS — современная операционная система, приятная глазу, удобная, интуитивно понятная, местами напоминающая macOS. Она полегче популярных Ubuntu и Mint, с минимальным набором установленных приложений. Дополнительныe можно устанавливать из тех же репозиториев, что Ubuntu и Mint.

Роли
  • Web – историк

    Специалист по анализу исторических хроник и событий в сфере IT. Главной задачей историка является изучение хронологических событий прошлого.

  • ГИС - специалист, картограф

    Специалист по составлению бумажных и электронных карт. Главной задачей картографа является составление карт и гидов, по достопримечательностям.

  • Сервисный инженер

    Cпециалист по обслуживанию и настройке программного и системного обеспечения. Главной задачей сервисного инженера является процесс установки, настройки и отладки работы ПО: системного и прикладного.

  • Оратор

    Данному участнику предстоит подготовить выступление и провести демонстрацию результатов прохождения Web – квеста.

  • IT - администратор

    Cпециалист по обеспечению штатной работы парка компьютерной техники, сети и программного обеспечения.
